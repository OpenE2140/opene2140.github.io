Built from the ground up

No longer tied to the limitations of the old era

  • Updated gameplay with modern features like attack-move, shift-queued orders, or unit stances.
  • Improved UI with easy-to-use tabbed production sidebar, useful UI overlays/hints, and the standard hotkeys for many actions (such as repairing buildings, selecting units by type, or issuing unit commands)
  • Many other, desperately-needed quality-of-life improvements (both visual and gameplay)
  • Native support for Windows, macOS and Linux

Based on a battle-tested RTS engine

Uses many features already provided by the OpenRA engine (kickstarting development)

  • Core engine features like input, rendering, UI, sounds, etc.
  • Core gameplay features like unit health, movement, pathfinding or weapons
  • Online and LAN multiplayer, replays, map editor
  • Extensibility, which allows modifying many aspects of the game
