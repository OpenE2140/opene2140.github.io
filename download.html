Linux

Any modern and recent enough Linux distribution is fine for OpenE2140. While OpenE2140 is developed mainly on Windows machines, we always test it on at least one Linux distribution (currently Ubuntu 24.04).

OpenE2140 is currently distributed on Linux system as AppImage only. We'll consider different distribution channels in the future.
